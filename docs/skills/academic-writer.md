## Writing Guidelines

- **Person**: Third person
- **Tense**: Present for methods; past for results
- **Tone**: Objective, measured, academic — no marketing language
- **Citations**: Numeric IEEE style [1], [2, 3]
- **Hedging**: "suggests", "indicates", "may" for uncertain claims
- **Every claim**: Backed by citation or experimental data

## Vocabulary Rules

| Avoid | Use Instead |
|-------|-------------|
| "game-changing" | "significant" |
| "revolutionary" | "novel" |
| "state-of-the-art" | "current best-performing" |
| "we believe" | "our results indicate" |
| "clearly shows" | "demonstrates" |
